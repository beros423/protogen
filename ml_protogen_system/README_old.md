# ML Protogen System

ML 모델 학습과 베이지안 최적화를 활용한 프로테인 발현 실험 설계 시스템입니다.

## 기능

### 1. 프로젝트 관리
- 단계별 프로젝트 진행 관리
- 왼쪽 사이드바를 통한 진행 상태 확인
- 각 단계별 데이터 입력 및 검증

### 2. 프로젝트 구조

#### 초기 설정 단계 (고정)
1. **프로젝트 설정**
   - 프로젝트 기본 정보 설정
   - 실험 유형 및 최적화 목표 설정
   - 예상 실험 규모 정의

2. **초기 데이터 입력**
   - 파일 업로드 (CSV, Excel)
   - 수동 데이터 입력
   - 템플릿 다운로드 지원

3. **초기 ML 모델 학습**
   - 다중 모델 학습 (Random Forest, Gradient Boosting, Linear Regression)
   - 교차 검증을 통한 모델 성능 평가
   - 최고 성능 모델 자동 선택

#### 반복 실험 사이클 (사용자가 원하는 만큼 반복)
1. **베이지안 최적화**
   - 현재 모델을 기반으로 베이지안 최적화 실행
   - 연속형 및 범주형 변수 동시 처리
   - 최적화 과정 시각화

2. **실험 설계**
   - 베이지안 최적화 결과 기반 실험 조건 제안
   - 추가 실험 조건 수동 입력 가능
   - 실험 선택 및 복제 설정

3. **매핑 파일 생성**
   - 96-well, 384-well 플레이트 지원
   - 매핑 파일 및 프로토콜 파일 자동 생성
   - 플레이트 레이아웃 시각화

4. **결과 입력 및 모델 업데이트**
   - 실험 결과 입력 (파일 업로드 또는 수동 입력)
   - 누적 데이터로 모델 재학습
   - 모델 성능 개선 추적

#### 프로젝트 완료
- **전체 프로젝트 요약**
  - 사이클별 성능 개선 그래프
  - 전체 실험 데이터 분석
  - 데이터 내보내기

### 3. 핵심 기술
- **머신러닝**: scikit-learn 기반 다중 모델 학습
- **베이지안 최적화**: bayesian-optimization 라이브러리 활용
- **데이터 시각화**: Plotly를 통한 인터랙티브 차트
- **UI**: Streamlit 기반 웹 인터페이스

## 설치 및 실행

```bash
cd ml_protogen_system
pip install -r requirements.txt
```

### 2. 애플리케이션 실행
```bash
streamlit run app.py
```

## 사용 흐름 예시

### 1단계: 프로젝트 시작
1. 프로젝트 이름 및 설명 입력
2. 초기 실험 데이터 업로드 (20-50개 샘플 권장)
3. 초기 ML 모델 학습

### 2단계: 첫 번째 사이클
1. **베이지안 최적화**: 현재 모델로 최적 조건 탐색
2. **실험 설계**: 5-10개 새로운 실험 조건 제안
3. **매핑 생성**: 실험 조건을 플레이트 레이아웃으로 변환
4. **실험 수행**: 실제 실험 진행 (시스템 외부)
5. **결과 입력**: 실험 결과를 시스템에 입력
6. **모델 업데이트**: 새 데이터로 모델 재학습

### 3단계: 추가 사이클 (필요한 만큼 반복)
- 각 사이클마다 모델이 개선되어 더 정확한 예측 제공
- 베이지안 최적화가 더 효율적인 실험 조건 제안
- 목표 성능에 도달하거나 더 이상 개선이 없을 때까지 반복

### 4단계: 프로젝트 완료
- 전체 사이클 성과 분석
- 최종 모델 성능 평가  
- 모든 실험 데이터 내보내기

## 주요 특징

1. **반복적 학습-최적화 사이클**: 초기 데이터로 모델 학습 후 베이지안 최적화로 새 실험 조건 제안
2. **누적 학습**: 각 사이클의 결과가 모델에 누적되어 성능이 지속적으로 개선
3. **유연한 사이클 관리**: 사용자가 원하는 만큼 사이클을 반복하거나 언제든지 종료 가능
4. **실시간 성능 추적**: 사이클별 모델 성능 개선을 실시간으로 시각화
5. **직관적인 UI**: 사이드바를 통한 진행 상황 확인 및 단계별 가이드
6. **자동화**: ML 모델 선택, 최적화, 매핑 파일 생성까지 자동 처리
7. **다양한 데이터 형식**: CSV, Excel 파일 업로드 및 수동 입력 지원

## 기존 protogen.py와의 차이점

- **적응형 실험 설계**: 고정된 실험 → 학습 결과 기반 적응형 실험 설계
- **반복적 개선**: 일회성 실험 → 지속적인 학습-실험-개선 사이클
- **지능형 최적화**: 수동 조합 → 베이지안 최적화 기반 지능형 실험 제안
- **성능 추적**: 단순 결과 → 사이클별 성능 개선 추적 및 시각화
- **확장성**: 고정 단계 → 무한 확장 가능한 실험 사이클
